# zood

zood 网页文档生成的 python 库, 可以将本地 Markdown 文件转为 Web 网页

zood 的页面风格更倾向于纯文档内容而非博客, 您可利用 Github Pages 为每一个仓库部署单独的网页文档

## 安装与使用

```bash
pip install zood
```

## 快速开始

```bash
zood init
zood -o
```
